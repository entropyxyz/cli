# Dev net CLI-TOOL

## Getting Started

### Install

- ` npm install`

- you can add a `.env` file or input env values when prompted by the cli

```bash
ENDPOINT_CHAIN=""
SEED=""
NAME=""
```

### Run Cli

- `npm run start`

This will bring up a list of options

The proper flow from start it to

1. create and entropy account
2. add the created seed to the `.env`
3. fund the account
4. register
5. sign

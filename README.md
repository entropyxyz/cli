# Devenet CLI-TOOL 

### Running 

* ``` yarn ```

* you can add am .env file or input the info when prompted by the cli 

```
ENDPOINT_CHAIN=""
SEED=""
NAME=""
```
* to start the CLI run: 
* ```yarn entropy```

The CLI currently has the following options:
1. Entropy Faucet
2. Balance
3. Register
4. Programs
5. Sign
7. Transfer
8. Give Zaps
9. New Entropy Wallet

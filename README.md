# Devenet CLI-TOOL 

### Running 

* ``` npm install```

* you can add a .env file or input the info when prompted by the cli 

```
ENDPOINTS_THRESHOLD=[""]
ENDPOINT_CHAIN=""
SEED=""
NAME=""
```

* ```npm run start```

This will bring up a list of options 

The proper flow from start it to 
1) create and entropy account 
2) add the created seed to the .env 
3) fund the account
4) register
5) sign
